## 摘要
大语言模型智能体在部分可观测的长期环境中运行，获取监督信息仍是一个主要瓶颈。我们通过利用现有后训练方法中忽视的一种监督来源来解决这一问题：嵌入在智能体运行轨迹中的意外却成功的子目标。具体来说，我们提出了事后监督学习（Hindsight Supervised Learning, HSL），其中辅助 LLM 审查每条已完成的轨迹，并用智能体实际达成的所有自然语言目标重新标记它。然后，HSL 将轨迹与其重标记的目标配对，并利用这些配对进行额外的微调。为了减轻重标记数据中的次优性，我们为 HSL 提出了两种学习技术：无关动作掩码和样本重加权。我们的实验表明，HSL 灵活且能与现有的后训练流程兼容。它同时改进了 SFT 和 DPO，在目标空间更多样化的长期任务上增益更大。此外，HSL 是样本高效的：在 ALFWorld 上，它仅使用四分之一的真实演示数据就超越了使用完整数据集训练的基线模型。

## Abstract
Large language model agents operate in partially observable, long-horizon settings where obtaining supervision remains a major bottleneck. We address this by utilizing a source of supervision overlooked in existing post-training methods: unintended yet successful goals embedded within agent rollouts. Specifically, we introduce Hindsight Supervised Learning (HSL), where an auxiliary LLM reviews each completed trajectory and relabels it with all of the natural-language goals the agent actually achieved. HSL then pairs the trajectory with its relabeled goals and uses these pairs for additional fine-tuning. To mitigate suboptimality in the relabeled data, we propose two learning techniques for HSL, irrelevant-action masking and sample reweighting. Our experiments show that HSL is flexible and compatible with existing post-training pipelines. It improves both SFT and DPO, with larger gains on long-horizon tasks with more diverse goal spaces. Moreover, HSL is sample-efficient: on ALFWorld, it surpasses baselines trained on the full dataset while using only one quarter of the ground-truth demonstrations.

## 2. 论文提出的方法论
*   **核心思想**：**事后监督学习（HSL）**。利用辅助大模型（LLM）在完整轨迹收集后，以“上帝视角”回溯并识别出智能体实际完成的**所有**有效子目标（即“成功演示”），并将原始轨迹与新目标配对进行监督学习，从而循环利用探索经验。
*   **HSL 三大核心步骤**：
    1.  **轨迹收集**：当前策略智能体在环境中执行原始指令，生成完整的交互轨迹。
    2.  **重标记过程**：
        *   **目标识别**：辅助 LLM（例如 Llama-3.3-70B）推理整个轨迹及最终观测，提取出智能体先后实现的**所有**自然语言目标列表。
        *   **动作相关性标注**：对于每个提取出的新目标，辅助 LLM 进一步判断轨迹中的每个动作是“相关”还是“无关”，生成一个二进制掩码序列。
    3.  **从重标记演示中学习**：
        *   将“轨迹片段-新目标-动作相关性标签”三元组存入动态缓冲区 `D'`。
        *   **无关动作掩码**（Irrelevant-action Masking）：在计算 SFT 损失时，屏蔽掉被标记为“无关”的动作，仅让模型模仿实现该目标所必需的关键步骤，从而训练一个更优的“事后专家策略”。
        *   **演示重加权**（Demonstration Reweighting）：依据解决任务的**难度**（用相关动作数量 `n_d` 代理）和解决的**最优性**（相关动作占比 `n_d/T_d`）为每条重标记演示分配权重 `w_d`，使模型更倾向于学习解决困难且解决方式更优的任务，以提升状态覆盖率。
*   **理论支撑**：通过 Theorem 1 推导了智能体策略与最优策略之间差异 `Δ(θ)` 的上界，该上界由专家演示拟合度、事后专家策略拟合度、覆盖率 `κ` 和事后专家最优性 `δ_E` 共同决定。这从理论上证明了在重标记演示上进行 SFT 能严格收紧差距（Corollary 1）。
*   **联合训练目标**：最终学习损失 `L_HSL` 是重标记数据集 `D'` 上的加权、掩码 SFT 损失与原始真实数据集 `D_train` 上的 SFT 损失的加权和（混合权重 `λ`）。

*   **组件有效性**：无关动作掩码和样本重加权对提升性能至关重要，能有效从次优轨迹中萃取出更优的专家行为路径。

## 7. 优点
*   **新颖且高效的监督挖掘范式**：相较于合成新数据或依赖环境奖励，巧妙地利用智能体自身的探索历史，将“失败”或“次优”的轨迹变废为宝，资源利用率高。
*   **理论结合实践的深度分析**：不仅提出了方法，还提供了严格的理论上界证明，并对重标记数据的分布、准确性、对各类目标的增益进行了量化分析，加深了对方法机理的理解。
*   **普适性与实用性**：HSL 框架与策略无关，展现出对主流后训练技术（SFT、DPO）的普适兼容性，易于集成到现有的 LLM 智能体训练流程中。

## 8. 不足与局限
*   **对强重标记模型的依赖**：目前高度依赖于一个强大但昂贵的外部 LLM（如 Llama-3.3-70B）进行零样本重标记。对于极端复杂的任务，可能需要进一步微调该重标记模型，增加了部署和推理成本。
*   **二元动作相关性标签过于简化**：假设动作仅分为“相关”与“不相关”。在更精细的任务中，动作可能“相关但次优”（例如，效率低下），简单的掩码无法指导模型收敛到全局最优策略。
*   **未完全解决探索多样性问题**：实验分析表明，重标记数据的分布仍会偏向训练集中出现频率高或容易“意外”达成的目标（如 ALFWorld 中的 `put`），长尾目标（如 `examine`）可能仍缺乏足够的重标记演示。
*   **依然需要少量真实演示**: 尽管高度样本高效，但 HSL 目前并未完全消除对真实演示的需求，这些演示用于提供基础的对齐和稳定训练，是其应用的一个前提条件。
*   **任务限定性**：目前的实验和应用场景均假设存在一个预定义的有效目标空间，尚未拓展到完全开放式的、无预定义目标的世界中。
