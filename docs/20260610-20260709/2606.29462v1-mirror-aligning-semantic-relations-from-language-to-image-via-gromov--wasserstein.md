## 摘要
多模态大语言模型（MLLMs）从其语言骨干中继承了丰富的关系先验，但当被要求在视觉上下文中应用这些关系时，却经常失败。我们将这种失败归因于一个结构性的盲点：基于投影的对齐训练每个视觉标记携带正确的语义，但从未询问概念间的关系是否在从语言到视觉的转换中得以保留。为了解决这个问题，我们提出MIRROR（通过基于最优传输的正则化将概念间关系从语言映射到视觉表示），这是一个几何正则化框架，通过利用语言表示中编码的丰富关系结构，将关系先验从语言迁移到视觉。具体来说，我们从所提出的半逆Gromov-Wasserstein（SI-GW）问题中导出一个代理损失，该问题是一个逆几何问题，旨在将视觉表示与源自语言的关系先验对齐。我们证明此公式存在唯一的闭式解，该解规定了由语言几何和跨模态耦合所暗示的理想视觉关系结构。公式的结构也使得计算高效，使其适用于长标记序列。在仅解码器Transformer中应用SI-GW需要精心设计。我们在层、头和标记级别引入针对性策略，以确保在没有额外参数或推理成本的情况下实现稳定的提取。MIRROR在保持通用视觉语言任务性能的同时，提高了关系一致性。

## Abstract
Multimodal Large Language Models (MLLMs) inherit rich relational priors from their language backbones, yet often fail when asked to apply these relationships in visual contexts. We trace this failure to a structural blind spot: projection-based alignment trains each visual token to carry the right semantics, but never asks whether the relationships between concepts survive the crossing from language to vision. To address this, we propose MIRROR (Mapping Inter-concept Relations from language to visual Representation via Optimal-transport-based Regularization), a geometric regularization framework that transfers relational priors from language to vision by exploiting the rich relational structure encoded in language representations. Specifically, we derive a surrogate loss from the proposed Semi-Inverse Gromov-Wasserstein (SI-GW) problem, an inverse geometric problem that aligns visual representations with language-derived relational priors. We show that this formulation admits a unique closed-form solution that prescribes the ideal visual relational structure implied by language geometry and cross-modal coupling. The structure of the formulation also enables efficient computation, making it applicable to long token sequences. Applying SI-GW inside decoder-only Transformers requires careful design. We introduce targeted strategies at the layer, head, and token levels to ensure stable extraction without additional parameters or inference cost. MIRROR improves relational consistency while preserving performance on general vision-language tasks.

### 1. 论文的核心问题与整体含义

- **研究动机**：多模态大语言模型（MLLMs）虽然在文本推理中继承了丰富的语义关系知识（如空间支持、类别差异等），但当需要将这类关系知识应用于视觉场景时，模型表现大幅下降（例如，文本可判断“坐”需要重心在支撑面上，但面对图片仍无法正确识别空间支持关系）。
- **核心问题**：现有主流MLLMs通过投影适配器（projection adapter）将视觉标记对齐到语言空间，并以语言建模目标训练。这种训练方式仅保证了**身份级别**的对齐——即每个视觉标记单独携带正确语义，却未顾及概念间的关系结构是否在跨模态过程中保留完好。因此，语言模型中原有的结构化关系知识在视觉使用时经常被忽略。
- **整体含义**：要提升多模态关系推理，不能仅关注“概念是什么”，还需关注“概念之间如何关联”，即需要对齐语言与视觉表征空间的**几何关系结构**。MIRROR正是从这一动机出发，提出了一种几何正则化方法，将语言中已编码的丰富关系先验迁移到视觉表征中。

### 2. 方法论

#### 核心思想
- 将跨模态关系对齐建模为一个**几何正则化问题**：语言内部的成对距离结构（Dt）应当通过跨模态注意力耦合（C）在视觉空间（Dv）中得到保持。
- 提出**半逆Gromov–Wasserstein（SI-GW）问题**：固定文本距离矩阵Dt和跨模态耦合C，求理想视觉距离矩阵（即最优的Dv），使得两空间的成对距离在C下保持一致。

#### 关键技术细节与公式
- **距离定义**：利用自注意力矩阵，经对称化与负对数变换得到文本和视觉的成对距离矩阵 Dt、Dv：`D = -log( (A+A^T)/2 + ε )`。
- **耦合来源**：跨模态注意力矩阵 C（来自中间LLM层的文本→视觉注意力），它提供了文本标记与视觉标记的软对应关系。
- **半逆GW问题**：  
  `bDv = argmin_{Dv} Σ_{i,j,k,l} (dt(i,j) - dv(k,l))^2 Cik Cjl`  
  并证明该问题具有唯一闭式解：  
  `bDv = (C^T Dt C) ⊘ (b b^T)`，其中 `b = C^T 1`。  
  该解可解释为：理想视觉距离是文本距离在耦合条件分布下的期望。
- **SI-GW损失**：使用F范数度量当前视觉距离Dv与理想距离bDv的差异：  
  `LSI-GW = ‖ Dv - bDv ‖_F^2`。  
  该损失的计算复杂度从原来的 O(nt² nv²) 降低至 O(nt² nv + nv² nt)。
- **稳定提取策略**（用于大规模Transformer应用）：
  - **层解耦**：文本自注意Dt取自早期层（关系较纯），视觉自注意Dv取自视觉编码器最后一层，耦合C取自中间层（信息浓度适中）。
  - **熵选头**：按跨注意力头的熵值排序，选择最低熵的 top-k 个头聚合为耦合矩阵 C，避免噪声。
  - **非语义标记抑制**：通过位置和注意力质量权重，对系统提示、图片占位符等无视觉语义标记进行软衰减，再计算 Dt。
- **整体训练**：`L_total = L_LM + λ * L_SI-GW`，无额外参数或推理开销。
