## 摘要
多模态大语言模型依赖上下文学习进行快速任务适应，但其可扩展性受到有限上下文窗口以及长多模态序列中键值缓存成本增长的严重制约。现有记忆压缩方法通常依赖于硬性地移除令牌或依赖于样本的重要性估计，这引入了偏差，破坏了语义结构，尤其是对于视觉表示而言，并产生无法适应新查询的静态记忆。我们提出TASM（任务感知结构化记忆），一个无需训练的框架，通过任务感知、结构保持和可动态访问的记忆构建来解决这些局限性。TASM采用任务向量引导的压缩，用捕捉跨示例共享相关性的任务级方向替换样本特定的信号。为保持底层流形，它通过二分图匹配应用语义感知的令牌合并，聚合令牌而不进行破坏性剪枝。最后，TASM将记忆结构化为包含紧凑核心记忆和潜在库的层次结构，便于查询自适应的动态检索。实验评估证实，TASM在高度压缩下保持高性能，有效平衡了效率与适应性。

## Abstract
Multi-modal large language models (MLLMs) depend on in-context learning (ICL) for rapid task adaptation, but their scalability is severely limited by finite context windows and the growing cost of key-value (KV) caches in long multi-modal sequences. Existing memory compression approaches typically rely on rigid token removal or sample-dependent importance estimation, which introduces bias, disrupts semantic structure, particularly for visual representations, and yields static memories that cannot adapt to new queries. We introduce TASM (Task-Aware Structured Memory), a training-free framework that addresses these limitations through task-aware, structure-preserving, and dynamically accessible memory construction. TASM employs task-vector guided compression to replace sample-specific signals with a task-level direction that captures shared relevance across demonstrations. To preserve the underlying manifold, it applies semantics-aware token merging via bipartite graph matching, aggregating tokens without destructive pruning. Finally, TASM structures memory into a hierarchy comprising a compact Core Memory and a Latent Bank, facilitating query-adaptive dynamic retrieval. Evaluations confirm TASM maintains high performance under heavy compression, effectively balancing efficiency with adaptability.

-   **现有方法的局限**：
    -   **样本特定偏差**：现有压缩方法（如基于注意力分数的剪枝）过度拟合于示例答案，移除了对测试查询至关重要的上下文。

-   **核心问题**：如何构建一种**任务感知的、保持结构完整性的、可动态访问的记忆机制**，以实现高效且鲁棒的多模态长上下文学习？

### 2. 论文提出的方法论

论文提出了一个名为 **TASM（Task-Aware Structured Memory，任务感知结构化记忆）** 的无需训练的框架，通过三大核心技术创新来改革记忆压缩过程。

#### 2.1 任务向量引导的压缩（Task-Vector Guided Compression）

-   **核心思想**：不再依赖特定样本的注意力分数，而是提取一个代表“问题到答案”转换方向的全局“任务向量”，并基于令牌在此方向上的投影来评估其重要性。
-   **关键技术细节**：
    -   **任务向量提取**：利用少量示例（Few-Shot Set），计算问题表示和答案表示在激活空间中的语义中心点，它们之间的归一化差向量即为**任务向量（τ）**。
    -   **正交投影评分**：通过计算缓存中每个令牌的键表示在任务向量上的投影，量化其任务相关性，从而**过滤掉与任务方向正交的噪声**。
    -   **层自适应门控**：考虑到 Transformer 浅层处理局部特征，深层处理全局语义，设计了一个基于层深的门控函数 λ(l)，用于在局部注意力分数和全局任务分数之间进行平滑插值，实现分层的重要性评估。

#### 2.2 语义感知的令牌合并（Semantics-Aware Token Merging）

-   **核心思想**：摒弃破坏性的硬性剪枝，将压缩视为一个**空间约束下的图匹配问题**，通过聚合而非丢弃信息来保持视觉拓扑结构。
-   **关键技术细节**：
    -   **图构建**：将高重要性令牌设为“汇聚节点”，低重要性令牌设为“源节点”，构建一个二分图。
    -   **空间约束兼容性**：边权重由令牌间的语义相似度和一个**空间正则化项（Ψ）** 共同决定，强制视觉令牌只能与其空间邻近的令牌合并，从而保持局部特征几何结构。
    -   **流形保持聚合**：被合并的“汇聚节点”的最终状态是其自身和所有分配来的“源节点”的加权中心，实现信息的特征压缩。

#### 2.3 信息几何驱动的动态检索（Information-Geometric Dynamic Retrieval）

-   **核心思想**：通过双层记忆架构和基于分布偏移的触发机制，实现查询驱动的自适应信息访问。
-   **关键技术细节**：
    -   **层次记忆架构**：将记忆分为 GPU 常驻的、高带宽的**核心记忆（Core Memory）** 和 CPU 卸载的、大容量的**潜在库（Latent Bank）**。
    -   **检索触发器**：使用**Jensen-Shannon（JS）散度**持续监测当前查询注意力分布与参考分布的差异。当差异超过阈值时，表明当前查询关注到了新的语义子空间，触发从潜在库中检索相关令牌。
    -   **稀疏检索机制**：当触发器激活时，对潜在库执行 Top-K 检索，将被激活的令牌动态注入到有效上下文中，其余时间则保持低复杂度的推理。
