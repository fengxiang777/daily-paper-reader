## 摘要
参数高效微调能以低成本使大语言模型适配单一任务，但在任务序列中，LoRA风格的方法会不断在相同的冻结权重上叠加低秩更新，导致每个新任务倾向于覆盖前序任务。我们提出ReCoLoRA（低秩适配器递归整合），一个面向持续微调的谱感知框架：适配器从预训练权重的随机奇异值分解初始化，每层的有效秩通过肘部准则选取，在释放残余容量前先适配主子空间。在每个新任务之前，ReCoLoRA将当前有效权重（而非原始权重）重新分解为一个冻结残差、一个缓慢更新的主成分和一个新的适配器（递归整合），从而每个任务都从已吸收前驱任务的模型开始。在四个7-8B基座模型上执行六任务持续GLUE序列时，与经过秩搜索的LoRA、PiSSA、AdaLoRA和DoRA基线相比，ReCoLoRA在三个基座模型上取得了最优的最终平均得分，同时训练更少的参数；一个基于预言路由的任务库变体作为完全任务隔离的上界。代码：https://github.com/bhqy666/ReCoLoRA。

## Abstract
Parameter-efficient fine-tuning adapts a large language model to one task cheaply, but across a task sequence LoRA-style methods keep stacking low-rank updates on the same frozen weight, so each new task tends to overwrite the previous ones. We present ReCoLoRA (Recursive Consolidation of Low-Rank Adapters), a spectrum-aware framework for continual fine-tuning: adapters are initialized from a randomized SVD of the pretrained weight, per-layer effective ranks are selected by an elbow criterion, and the principal subspace is adapted before residual capacity is opened. Before each new task, ReCoLoRA re-decomposes the current effective weight, rather than the original one, into a frozen residual, a slowly updated principal component, and a fresh adapter (recursive consolidation), so every task starts from the model that has already absorbed its predecessors. On a six-task continual GLUE sequence over four 7-8B backbones, ReCoLoRA attains the best final average score on three of the four backbones against rank-swept LoRA, PiSSA, AdaLoRA, and DoRA baselines while training fewer parameters; an oracle-routed task-bank variant serves as an upper bound under full task isolation. ### 2. 论文提出的方法论
- **核心思想**：利用预训练权重的奇异值谱，在适配新任务时优先更新对模型结构影响大的“主成分”方向，并将完成的任务递归地“固结”回模型的慢变部分，避免后续更新冲刷掉先前知识。
- **关键技术细节**：
    - **随机 SVD 初始化**：使用随机化 SVD 对预训练权重矩阵进行高效分解，将适配器的初始化方向对齐到主子空间。
    - **肘部准则选秩**：根据奇异值的累积能量比值（$\rho$ = 0.8）自动为每层选择“有效秩” $r^*$，对高能量方向分配更多容量，低能量方向视为残余，以此控制可塑参数的数量。
    - **两阶段训练**：
        - **第一阶段**：仅训练主适配器分支，在稳重的预训练主方向上学习任务结构。
        - **第二阶段**：逐步释放残余容量 $\gamma(t)$，让模型获取额外的自由度，但采用更小的学习率，避免剧烈扰动。
    - **递归整合（核心机制）**：每完成一个任务后，不再以原始 $W_0$ 为基础。而是**将当前的有效权重 $W_{eff}$ 重新 SVD 分解**为三个部分：
        1.  冻结的残差分量 $W_{res}$
        2.  缓慢更新的主成分分量 $W_{slow}$（以极低学习率微调，承载历史知识）
        3.  新的快速适配器 $W_{fast}$（承担新任务的可塑性）
        这样下个任务开始时，模型已经整合了之前所有任务的知识。
    - **TaskBank 变体（上界）**：作为对照，为每个任务隔离训练一个独立的谱初始化适配器分支，并在推理时使用预言路由（Oracle Routing），从结构上彻底消除任务间的覆盖。

### 6. 论文的主要结论与发现
- **谱感知显著提升单适配器效果**：静态 ReCoLoRA 在 Qwen3-8B 上就以远少于秩-256 基线的参数量，获得了最高平均分和更低遗忘，证明谱初始化和两阶段训练能提升单适配器的质量和抗干扰能力。
- **递归整合是构建可演进模型的关键**：相比将所有任务硬冻结或堆叠在原始 $W_0$ 上，递归整合在 Qwen3、Mistral 和 InternLM2.5 三个基座上均取得最优的最终平均分，且参数量最少，成功将历史知识融入一个模型中。

- **方法存在骨干敏感性**：递归 ReCoLoRA 在 Llama-3.1-8B-Instruct 上并未超越标准 LoRA，表明方法的优势并非在所有模型上一致，可能依赖模型预训练权重的谱特性。

### 7. 优点
- **创新的持续微调范式**：不依赖数据回放、不堆叠独立适配器，通过“递归整合”使模型持续内化知识，更接近人类持续学习的过程。
- **高效的参数控制**：通过肘部准则实现每层自适应秩分配，在取得更好效果的同时使用了比大多数基线更少的训练参数。
- **清晰的原理剖析**：将遗忘问题解耦为“如何学好任务”（谱初始化）和“如何避免覆盖”（递归整合/隔离），并通过 TaskBank 上界实验清晰地展示了这两部分的贡献上限。
- **扎实的实验验证**：在多个主流大模型上进行评估，包含严谨的消融和敏感性分析，增强了方法的说服力。

### 8. 不足与局限
- **骨干敏感性未解决**：递归整合在 Llama 上未取得优势，说明方法对特定模型权重谱结构的依赖性较大，移植性受到限制。

- **超参数可能需按模型调整**：SVD 截断、肘部阈值、慢更新学习率等敏感参数可能需要针对不同模型族进行调优，论文未提供通用配置。
