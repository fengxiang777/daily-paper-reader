## 摘要
视觉语言模型（VLMs）在各类任务中展现出卓越的能力，但其计算成本主要受馈入语言模型的大量视觉令牌所支配。现有的令牌缩减方法依赖于基于注意力的分数或成对相似性，缺乏对每个令牌的显式语义表示。我们提出了TORINO（通过可解释概念重叠进行令牌缩减），这是一种即插即用的框架，用于VLM中自适应的视觉令牌缩减，无需对底层模型进行微调。TORINO利用稀疏自编码器（SAEs）将视觉令牌投影到可解释的潜在空间，在该空间中，令牌关系可通过共享的概念激活加以分析。具体而言，我们将概念重叠定义为活跃SAE潜变量之间的一致程度，并利用它来分组共享语义内容的令牌。随后通过剪枝或合并对每组内的令牌进行缩减，从而提供一个统一的框架，在去除冗余的同时保留语义上重要的视觉信息。与固定预算的方法不同，TORINO能根据输入复杂度动态调整缩减率，使不同图像得以保留不同数量的令牌。在多个视觉语言基准上的实验表明，TORINO实现了优越的效率-准确度权衡，在将性能损失降至最低的前提下减少了视觉令牌的数量。

## Abstract
Vision-Language Models (VLMs) have demonstrated impressive capabilities across different tasks, but their computational cost is dominated by the large number of visual tokens fed to the language model. Existing token reduction methods rely on attention-based scores or pairwise similarity, without an explicit semantic representation of each token. We introduce TORINO (TOken Reduction via Interpretable coNcept Overlap), a plug-and-play framework for adaptive visual token reduction in VLMs that requires no fine-tuning of the underlying model. TORINO leverages Sparse Autoencoders (SAEs) to project visual tokens into an interpretable latent space where token relationships can be analyzed through shared concept activations. Specifically, we define concept overlap as the degree of agreement between active SAE latents and use it to group tokens that share semantic content. Reduction within each group is then performed by either pruning or merging, providing a unified framework that preserves semantically important visual information while removing redundancy. Unlike fixed-budget approaches, TORINO dynamically adapts the reduction rate to input complexity, allowing different images to retain different numbers of tokens. Experiments across multiple vision-language benchmarks show that TORINO achieves favorable efficiency-accuracy trade-offs, reducing the number of visual tokens with minimal performance loss.

*   **核心问题**：现有视觉令牌削减方法（如剪枝、合并）主要依赖注意力分数、CLS 令牌相似性或原始嵌入距离来决定令牌的重要性，缺乏对每个令牌所编码概念的**显式语义表示**。这可能导致视觉上不相似但语义上冗余的令牌被错误保留，反之亦然。
*   **整体含义**：本文探索如何利用稀疏自编码器（SAE）的可解释、单义性特征来引导令牌削减决策，提出一个**即插即用、无需微调、动态自适应**的框架 TORINO，根据图像内容语义来决定保留或合并令牌，从而更好地平衡模型效率与准确性。

### 2. 论文提出的方法论
TORINO 操作于 VLM 视觉编码器的某一中间层，流程分为三阶段：
*   **SAE 特征提取**：
    *   从冻结的视觉编码器（如 CLIP ViT-L/14）的特定层（如第 22 层）提取 N 个令牌嵌入 \(v_i\)。
    *   通过一个预训练的 Matryoshka BatchTopK 稀疏自编码器（SAE）将其编码为稀疏、非负的概念向量 \(z_i = \phi(v_i)\)，向量的每个维度对应一个可解释的潜在概念，激活值 \(z_{ij}\) 表示令牌 i 对概念 j 的响应强度。
*   **概念引导的令牌分组**：
    *   对于每个令牌，选取其 k 个最大激活值对应的概念索引构成集合 \(A_i\)。
    *   构建令牌图：若两个令牌的 top-k 概念集合的交集大小 ≥ 阈值 δ（即概念重叠度），则存在一条边。
    *   图的每个连通分量即为一个概念组 \(\{G_g\}\)，组数 \(G_{dyn}\) 由图像内容动态决定（语义单一图像组少而大，复杂图像组多而小）。
*   **令牌削减**：
    *   **剪枝模式 (TORINO-P)**：每组保留峰值激活值 \(a(i)=\|z_i\|_{\infty}\) 最高的令牌，丢弃其余。
    *   **合并模式 (TORINO-M)**：将组内所有令牌取平均并用 \(1+\log|G_g|\) 做对数缩放调整，得到一个虚拟令牌。
    *   **固定预算变体**：引入截断（保留最大的 B 个组）和填充（从余下令牌中选高激活值令牌补充到 B 个），以适配需固定输出长度的场景。

### 6. 论文的主要结论与发现
*   **效率-准确度权衡最优**：在中等压缩率（↓62%-78%）下，TORINO 的表现优于所有基线方法，能以最小的性能损失换取大量令牌削减（如 98.7% 和 97.1% 的相对分数）。
*   **内容自适应性有效**：不同复杂度图像保留的令牌数存在显著差异（如 80% 个案跨度达 20-25pp），无需显式预算控制即可自发适应。
*   **剪枝与合并的互补性**：低到中压缩时合并（TORINO-M）略优，高压缩时剪枝（TORINO-P）鲁棒性更好，因对数缩放难以弥补大组带来的信息损失。
*   **SAE 的语义优势**：利用 SAE 可解释概念分组比直接基于原始嵌入分组能有效提升性能，证实了语义信息在令牌削减中的价值。
*   **字典规模的影响**：极大的 SAE 扩张因子（ε=64）因其高单义性，在极端压缩（↓92%）下成为了唯一保持鲁棒性的配置。

### 7. 优点
*   **新颖性与可解释性**：首次将机械可解释性工具（SAE）用于驱动功能性决策（令牌削减），为“令牌重要性”提供了显式的、基于概念的评估标准。
*   **即插即用**：无需对底层 VLM 进行任何微调，直接作用于推理时，易集成。
*   **动态自适应**：天然的输入自适应性，能根据图像语义丰富度自动决定保留的令牌数量，无需额外的复杂度估计模块。
*   **统一框架**：通过改变组内处理方式，自然统一了剪枝和合并两种策略，便于比较和分析。
*   **实证扎实**：在多个模型和广泛基准上进行了详尽的定量、定性和消融分析，结论可靠。

### 8. 不足与局限
*   **SAE 训练的局限性**：所使用的 SAE 仅在 CLS 令牌上训练，尽管在深层表现尚可，但实验表明其在空间令牌上训练的 SAE 效果显著下降，这可能限制了其捕捉细粒度局部概念的能力。
*   **高压下的小对象风险**：在极高的压缩率下，空间上孤立的小对象可能构成一个很小的概念组，极易被整个裁剪掉，导致模型无法回答相关问题，这是内容自适应方法的固有缺点。
*   **模型架构依赖**：当前评估仅限于 LLaVA 模型（使用 CLIP ViT-L/14）。若要应用到其他 VLM 骨干网络，需要为该特定编码器重新训练一个匹配的 SAE，增加了使用门槛。
*   **分组是查询无关的**：当前的概念分组基于整张图像，不依赖于用户具体的提问。这可能导致保留了与问题无关的语义区域，而未优先保留对回答问题至关重要的区域。
