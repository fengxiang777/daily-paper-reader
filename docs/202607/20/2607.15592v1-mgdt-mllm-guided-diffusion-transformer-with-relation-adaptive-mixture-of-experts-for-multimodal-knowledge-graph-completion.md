## 摘要
多模态知识图谱补全（MKGC）需要从结构、文本和视觉线索中推断缺失实体。现有的基于扩散的MKGC方法通常直接在原始多模态特征上去噪，这一设计迫使去噪器同时执行关系依赖的线索选择、跨模态语义对齐和结构感知实体生成，从而为扩散过程引入噪声和语义不一致的条件，导致补全性能不佳。针对这一局限，我们提出MGDT——基于关系自适应混合专家的MLLM引导扩散Transformer，这是一种基于“先对齐后扩散”范式的新型MKGC框架。MGDT首先采用关系自适应语义路由混合专家（RASR-MoE）模块，选择与关系相关的多模态语义转换路径并抑制无关模态的干扰；随后，使用冻结的多模态大语言模型（MLLM）作为语义锚点，将路由后的多模态表示对齐至统一潜在空间，以降低跨模态语义异质性；最后，知识图谱扩散Transformer（KGDT）在对齐空间中进行图条件去噪生成，获得缺失实体表示。在三个基准数据集上的实验表明，MGDT始终优于强基线模型。

## Abstract
Multimodal Knowledge Graph Completion (MKGC) requires inferring missing entities from structural, textual, and visual cues. Existing diffusion-based MKGC methods usually denoise directly on raw multimodal features. Such a design forces the denoiser to simultaneously perform relation-dependent cue selection, cross-modal semantic alignment, and structure-aware entity generation, which introduces noisy and semantically inconsistent conditions for diffusion and consequently leads to suboptimal completion performance. To address this limitation, we propose MGDT: MLLM-Guided Diffusion Transformer with Relation-Adaptive Mixture-of-Experts (MGDT), a novel MKGC framework built on an align-then-diffuse paradigm. MGDT first employs a Relation-Adaptive Semantic Routing Mixture-of-Experts (RASR-MoE) module to select relation-relevant multimodal semantic transformation paths and suppress irrelevant modality interference. MGDT then uses a frozen Multimodal Large Language Model (MLLM) as a semantic anchor to align the routed multimodal representations into a unified latent space and reduce cross-modal semantic heterogeneity. Finally, a Knowledge Graph Diffusion Transformer (KGDT) performs graph-conditioned denoising generation in the aligned space to produce the missing entity representation. Experiments on three benchmark datasets show that MGDT consistently outperforms strong baselines.

  - **关系依赖的多模态线索选择**：不同关系依赖不同的模态信号（如图结构、文本或视觉），固定的多模态融合会引入大量与当前关系无关的噪声。
- **现有方法局限**：当前扩散式MKGC方法往往直接在原始多模态特征上去噪，迫使去噪模型在同一个过程中同时处理线索选择、语义对齐和实体生成，容易产生噪声条件和语义不一致，导致补全效果次优。
- **本文整体含义**：提出“先对齐后扩散”的新范式（M2GDT），通过解耦多模态线索选择、语义对齐和结构感知生成三个步骤，在关系自适应、语义对齐的潜在空间中执行扩散生成，从而显著提升MKGC的性能。

## 2. 论文提出的方法论

M2GDT框架包含三个核心组件：关系自适应语义路由混合专家（RASR-MoE）、基于冻结MLLM的语义锚对齐、知识图谱扩散Transformer（KGDT）。整体流程如下：

- **关系自适应语义路由混合专家 (RASR-MoE)**  
  - 对实体ℎ在结构、视觉、纹理三个模态上分别获取嵌入 \(x_s, x_v, x_t\)，并引入关系嵌入 \(r\)。  
  - 每个模态设有 \(M\) 个专家，通过关系条件路由器计算权重 \(\tilde{g}_k = \text{softmax}(\text{MLP}_k([x_k; r]) + \delta_k)\)，其中 \(\delta_k\) 为高斯噪声增强鲁棒性。  
  - 各专家输出经过加权求和得到路由模态表示 \(\tilde{e}_k\)，再通过模态专属投影矩阵统一到共享空间，并通过逐元素相加得到融合表示 \(e_f = e_s + e_v + e_t\)。  
  - 额外引入辅助监督损失 \(L_{moe}\) 对结构、视觉、纹理和融合四个分支分别计算二元交叉熵，以保留判别力；同时加入轻型CLUB风格正则项 \(L_{reg}\) 鼓励专家互补。

- **MLLM锚定语义对齐**  
  - 将实体的原始多模态特征拼接后经可学习投影送入冻结的Qwen3-VL语言骨干，获得高维隐藏状态 \(h_{anchor}\)。  
  - 从 \(h_{anchor}\) 经不同投影头得到与MoE编码器输出维度匹配的锚特征 \(a_s, a_v, a_t, a_f\)。  
  - 对结构、视觉、纹理分支采用余弦相似度对齐损失 \(L_k = 1 - \frac{1}{B}\sum \cos(e_{k,n}, a_{k,n})\)，对融合分支使用温度缩放KL散度匹配损失 \(L_f = \frac{1}{B}\sum \text{KL}(p_{f,n} \parallel q_{f,n})\)。总对齐损失为 \(L_{align} = \lambda_k L_k + \lambda_f L_f\)（结构、视觉权重设为1.0，其他为辅助正则项）。

- **知识图谱扩散Transformer (KGDT)**  
  - 将对齐后的多模态表示作为扩散条件 \(e_f\)，与时间步嵌入 \(\tau_t\) 相加构成条件嵌入 \(c_t\)。  
  - 采用标准DDPM前向过程：\(z_t = \sqrt{\bar{\alpha}_t} z_0 + \sqrt{1-\bar{\alpha}_t} \epsilon\)。  
  - 反向过程由Transformer骨干的KGDT实现：\(\hat{\epsilon} = \text{KGDT}(z_t, c_t; \Theta)\)，通过交叉注意力注入条件，堆叠多个Transformer块逐步去噪生成缺失实体表示。  
  - 总训练目标：\(L_{total} = \lambda_1 L_{moe} + \lambda_2 L_{reg} + \lambda_3 L_{align} + \lambda_4 L_{diff}\)，其中 \(\lambda_1=1.0, \lambda_2=0.0001, \lambda_3=0.01, \lambda_4=0.1\)。  
  - 推理时，从高斯噪声出发迭代去噪，将预测实体嵌入与候选实体嵌入对比进行排序。

## 7. 优点

- **范式创新**：明确解耦了多模态线索选择、语义对齐和实体扩散生成，改变了先前扩散方法将所有任务混杂于原始特征空间的局限。
- **关系自适应路由**：利用MoE动态选择关系相关的模态转换路径，并通过辅助监督和正则化增强专家多样性，有效抑制无关模态干扰。
- **MLLM作为语义锚点**：首次将冻结的多模态大模型引入知识图谱补全的对齐过程，利用其强大的跨模态先验为语义校准提供可靠依据，且无需微调大模型，训练开销可控。
- **全面的实验验证**：在多个数据集、多个基线、多维度的消融与敏感性分析中均呈现一致提升，论证充分。
- **理论依据与实证结合**：除了性能对比，还通过相似度动态监控和解耦分析揭示了方法的有效区间（适度对齐）和失效边界。

## 8. 不足与局限

- **算力消耗**：尽管大模型参数冻结，但推理和训练仍需依赖大型MLLM（如Qwen3-VL），可能对计算资源要求较高，尤其在更大规模知识图谱上的可扩展性未作深入讨论。
- **数据集规模**：实验仅限于三个中等规模的多模态知识图谱数据集（如DB15K约15K实体），在更大、更嘈杂的工业级知识图谱上的效果和效率尚待验证。
- **对齐权重的启发式设定**：结构、视觉分支的对齐权重被简单设为人1.0，缺乏自适应或可学习的调节机制，可能在模态不平衡时不够鲁棒。

- **任务形式限制**：针对尾实体预测（ℎ,?,?）对称处理，未深入探讨更复杂的图结构补全（如缺失三元组中间实体或关系），且仅关注实体嵌入层面的生成，缺乏对真实文本/图像输出的生成能力。
