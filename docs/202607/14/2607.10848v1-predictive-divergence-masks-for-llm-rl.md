## 摘要
大语言模型的强化学习通常依赖信任区域掩码来稳定离策略更新。主流的PPO风格方法使用采样令牌的重要性比率进行两个判别：一个邻近性判别，询问策略是否偏离行为策略太远；一个方向性判别，询问更新是否将其推得更远。最近的工作DPPO通过用行为策略与训练策略之间的概率散度取代PPO基于比率的测试，改进了邻近性判别。然而其方向性判别仍继承自PPO。仅当采样令牌的重要性比率偏离1时，令牌才会被掩蔽。我们观察到这种基于比率的方向性判别是一种单样本代理，其符号可能与定义邻近性判别的散度变化不一致。因此我们提出预测性散度掩码，它询问下一个策略梯度步骤是否会增加或减少信任区域所使用的相同散度。针对LLM强化学习中使用的离散softmax策略，我们以闭式推导了这一预测。由于生产环境的 rollout 引擎仅暴露词汇表的截断（top-K）视图，我们为该预测开发了两种轻量级 top-K 估计器。详细分析表明，基于散度的方向判别比采样比率更符合散度的实际变化，由此得到的掩码在不同模型规模和精度设置下都能改善强化学习训练。

## Abstract
Reinforcement learning for large language models (LLMs) typically relies on trust-region masks to stabilize off-policy updates. The dominant PPO-style approach uses the sampled-token importance ratio for two criteria: a proximity criterion, which asks whether the policy has moved too far from the behavior policy, and a direction criterion, which asks whether the update pushes it farther away. Recent work DPPO improves the proximity criterion by replacing PPO's ratio-based test with a probability divergence between the behavior and training policies. However, its direction criterion is still inherited from PPO. A token can be masked only when the sampled-token importance ratio moves away from one. We observe that this ratio-based direction criterion is a single-sample proxy that can disagree in sign with the change of the divergence that defines the proximity criterion. We therefore propose the predictive divergence mask, which asks whether the next policy-gradient step will increase or decrease the same divergence used by the trust region. For the discrete softmax policies used in LLM RL, we derive this prediction in closed form. Because production rollout engines expose only a truncated (top-K) view of the vocabulary, we develop two lightweight top-$K$ estimators for this prediction. Detailed analysis shows the divergence-based direction is better aligned with the realized change of the divergence than the sampled ratio, and the resulting masks improve RL training across model scales and precision settings.

## 1. 论文的核心问题与研究背景

- **问题背景**：在大语言模型（LLM）强化学习（RL）中，由于训练策略与数据生成策略（行为策略）存在训练-推理错配以及策略陈旧等问题，训练本质上是离策略的。主流的PPO方法通过**信任区域掩码**稳定更新，其掩码由两个准则构成：
  - **邻近准则**：判断训练策略与行为策略的偏差是否超出阈值。
  - **方向准则**：判断当前更新是否进一步增加偏差，仅当更新是“朝外”的才予以屏蔽。
- **现有问题**：PPO的两个准则都基于**采样token的重要性比率** \(r_t\)。近期的DPPO方法将邻近准则替换为分布间散度（如KL散度），但**方向准则仍继承PPO**——仅依赖单样本比率的符号 \(sign(\hat{A}_t(r_t-1))\)。作者指出，当信任区域由散度定义时，比率的符号可能与散度的实际变化方向不一致。这一单样本代理无法反映分布整体移动，导致方向判别不可靠，进而影响训练稳定性。
- **研究动机**：提出一种与散度邻近准则保持一致的方向准则，即直接预测策略更新是否会使同一散度增大，作为掩码判断依据。

## 2. 方法论

### 核心思想
将方向准则从“更新是否让比率离 1 更远”转化为“更新是否会使散度 \(D_t\) 增大”。通过计算散度沿策略梯度步的**方向导数**，以其符号作为屏蔽依据。

### 关键技术细节
- **分散度KL的方向导数（闭式解）**：对于 softmax 策略，令采样 token 为 \(k\)，则沿梯度方向 \(v_i = \mathbb{1}[i=k] - \pi_i\) 的KL散度变化的一阶近似为：
  
  \[
  \dot{D} = (\pi_k - \mu_k) + \sum_i \pi_i (\mu_i - \pi_i)
  \]
  
  分解为**局部项**（采样token处的概率差）和**全局项**（softmax归一化耦合导致的分布整体移动）。比率方向准则仅对应局部项，而全局项是其缺失部分。掩码决策仅依赖符号 \(sign(\hat{A}\cdot \dot{D})\)，与步长或比率大小无关。

- **截断词汇表下的估计**：生产环境仅暴露 top-K 概率和尾质量，提出两种轻量估计器：
  - **聚合尾部估计器**：将 top-K 之外的 tokens 合并为一个聚合项，\(\dot{D}_{\text{agg}}\)。
  - **均匀尾部估计器**：将尾部分布均匀散布到所有缺失 tokens 上，\(\dot{D}_{\text{uni}} \approx \dot{D}_{\text{agg}} \times \frac{1}{n-m}\)（\(n\) 为词表大小，\(m\) 为 top-K 大小）。当 \(n \gg m\) 时，两者差异极小。
- **预测散度掩码**：将更新后邻近准则与方向准则统一成：
  
  \[
  M_t^{\text{pred}} = 
  \begin{cases}
  0, & \text{if } sign(\hat{A}_t \cdot \dot{D}_t) > 0 \text{ and } D_t > \delta \\
  1, & \text{otherwise}
  \end{cases}
  \]
  
  仅在当前散度已超出阈值且预测更新会进一步增加散度时屏蔽该 token 的梯度。该掩码不引入任何新超参数。

## 3. 实验设计

- **训练数据**：过滤版 DAPO-Math-17k（约 13k 数学题）。
- **评测基准**：AIME24 和 AIME25，每题采样 16 次取平均准确率（avg@16）。
- **基模型与精度设置**：
  - Qwen3-4B-Base
  - Qwen3-8B-Base
  - Qwen3-30B-A3B-Base（两种低精度设置：FP8 仅用于 rollout，以及 FP8 全程训练+rollout）
- **对比方法**：
  - GRPO with clip-higher（ϵ_low=0.2, ϵ_high=0.28）
  - DPPO-TopK-KL（δ=0.15 或 0.05，K=20，邻近准则为 top-K KL，方向准则沿用比率）
  - 本文方法：DPPO-TopK-KL + 预测性散度掩码（聚合尾部估计器和均匀尾部估计器）
- **其他分析实验**：令牌级别的方向准则分析，使用 61 个随机种子，比较两种方向准则在已超出阈值 tokens 上的安全保留率。

## 6. 主要结论与发现

- 预测性散度掩码在所有模型规模和精度设置下均**一致优于** DPPO-TopK-KL 基线，提升了 AIME 平均准确率，且保持了稳定。
- GRPO clip-higher 在两个 30B 模型上出现不稳定甚至崩溃，而散度系列方法始终稳定。
- 令牌级别分析显示：在两种方向准则产生分歧的 tokens 上，**基于散度的方向准则**的“误保留（keep-but-diverged）”率更低（34.2% vs 36.9%），保留更新中散度实际收缩的比例更高（65.1% vs 62.9%），证实其与散度真实变化的对齐更佳。
- 聚合尾部与均匀尾部估计器表现非常接近，方法对尾部建模不敏感。

## 7. 优点

- **理论一致性**：使信任区域掩码的邻近准则与方向准则统一于同一散度度量，弥补了 DPPO 遗留的不一致。
- **轻量可实现**：方向导数仅依赖已有 top-K 概率和尾部质量，无需额外前向/后向传播，计算开销忽略不计。
- **强实证支持**：在多尺度、多精度、不同阈值设置下均稳定提升，且 micro-level 分析验证了其机制有效性。
- **可扩展性强**：给出了适用于 top-K KL 和 top-K TV 两种散度的推导与掩码形式。

## 8. 不足与局限

- **局部一阶近似**：掩码仅基于单个 token 的梯度方向预测散度变化，但实际参数更新汇总整个批次所有 tokens 的梯度，可能产生 token 间交互效应，预测并非完全精确。
- **任务单一性**：实验仅在数学推理任务（DAPO-Math）和 AIME 评测上进行，未涉及其它领域（如对话、代码生成等），方法的泛化性尚需验证。
- **依赖 top-K 统计**：方法假设 rollout 引擎已提供 top-K 日志概率和尾质量，对于不暴露这些信息的部署环境需要额外适配。
- **超参数 δ 仍需调节**：虽然不引入新超参数，但信任区域阈值 δ 仍会影响最终性能，tight 阈值（0.05）下所有方法均表现下降，方法对 δ 仍有敏感性。
