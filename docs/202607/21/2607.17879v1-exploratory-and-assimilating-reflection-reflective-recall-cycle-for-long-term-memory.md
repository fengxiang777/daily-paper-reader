## 摘要
基于大语言模型的自主智能体需要外部记忆来克服其无状态性和有限的上下文窗口，以实现长期交互和动态知识推理。然而，现有的记忆检索方法往往缺乏自适应性和样本效率，且难以从异构存储中检索出恰当的记忆组合。我们提出了探索性-同化性反思（EAR）框架，以实现高初始检索性能和样本高效的自适应。EAR结合了两种机制：探索性反思，通过迭代搜索来引导检索并为每个查询收集有用的经验；同化性反思，从经验缓冲区中回放这些经验，以比仅依赖即时奖励的方法更高效地优化全局重排序器。实验表明，在两个长期对话基准测试上，EAR将检索性能相较于基线检索器提升了高达17.9%。我们还表明，EAR具有高度的样本效率，并对噪声反馈具有鲁棒性。

## Abstract
LLM-based autonomous agents require external memory to overcome their statelessness and limited context window for long-term interaction and dynamic knowledge reasoning. However, existing memory retrieval methods often lack adaptability and sample efficiency, and struggle to retrieve the right mixture of memories from heterogeneous stores. We propose Exploratory-Assimilating Reflection (EAR), a framework for high initial retrieval performance and sample-efficient adaptation. EAR combines two mechanisms: Exploratory Reflection, which performs iterative search to bootstrap retrieval and collect useful experiences for each query, and Assimilating Reflection, which replays these experiences from an Experience Buffer to refine a global reranker more efficiently than methods relying only on immediate rewards. Experiments show that EAR improves retrieval by up to 17.9% over the baseline retriever on two long-term dialogue benchmarks. We also show that EAR is highly sample-efficient and robust to noisy feedback.

## 2. 方法论
### 整体框架：探索性-同化性反思（EAR）
EAR由两个协同工作的模块构成：
- **探索性反思**：模拟“工作记忆”，对每次查询执行多轮主动搜索，在候选记忆集中挑选出兼具相关性、多样性和探索价值的记忆子集，并收集LLM反馈作为经验。
- **同化性反思**：模拟“长期记忆”，将这些查询级别的探索经验存入**经验缓冲区**，通过相似查询回放来更新一个全局**残差适配器**，从而在多次查询中持续提升检索质量。

### 关键技术细节
#### 残差检索适配器
在冻结的原始嵌入基础上，学习两个残差变换矩阵 $W_q, W_m$，生成适配后的查询和记忆嵌入：
$$\tilde{\mathbf{q}} = \frac{(\mathbf{I} + W_q)\mathbf{q}}{\|(\mathbf{I} + W_q)\mathbf{q}\|_2}, \quad \tilde{\mathbf{m}}_i = \frac{(\mathbf{I} + W_m)\mathbf{m}_i}{\|(\mathbf{I} + W_m)\mathbf{m}_i\|_2}$$
适配器输出一个可学习的检索分布 $p_\phi$，作为后续探索与更新的基础。

#### 探索性反思：基于乌科班迪（UCB）的迭代选择
将候选记忆视为局部臂，每轮选择时综合考虑：
$$A_i^{(t,j)} = \lambda_{\text{rel}} \tilde{\mathbf{q}}^\top \tilde{\mathbf{m}}_i - \lambda_{\text{div}} D_i^{(t,j)} + \lambda_{\text{exp}} U_i^{(t)}$$
其中 $U_i^{(t)}$ 为UCB探索奖励，$D_i^{(t,j)}$ 为与已选记忆的冗余度惩罚。每次选出 $s$ 条记忆后获取批评家反馈（+1/-1），并更新UCB统计量。

#### 同化性反思与经验回放
将一次查询中每条候选记忆的平均反馈存储为经验 $e^{(n)}$。训练时，检索与当前查询最相似的若干条历史经验，用当前适配器重新计算损失，并结合当前查询损失进行联合梯度更新：
$$\mathcal{L}^{(n)}_{\text{EAR}} = \mathcal{L}^{(n)}_{\text{cur}} + \lambda_{\text{rep}} \mathcal{L}^{(n)}_{\text{rep}}$$
损失函数采用反馈加权的排序目标，增大正样本概率、降低负样本概率。

#### 推理模式
- **仅适配器模式**：直接根据适配后的相似度选 Top-$s$，无需额外批评家调用。
- **探索器辅助模式**：启用迭代探索，以额外的LLM调用换取更高检索精度。
