## 摘要
视觉语言模型（VLMs）通过思维链（CoT）提示实现了强大的推理能力，但会带来高昂的顺序生成成本、误差累积和有限的自纠正能力。扩散多模态大语言模型（dMLLMs）以顺序无关的方式解码令牌，提升了效率并支持迭代优化，但它们的推理能力及其增强途径尚未被充分探索。我们提出了一种无需训练的方法——时空令牌否决（ST-Veto），它利用了在每个扩散步骤中能够观察所有令牌位置的能力。ST-Veto不仅依赖当前步骤的置信度，还通过置信度动态的二阶泰勒预测来否决时间上不稳定的令牌，并利用图像注意力质量过滤弱定位的令牌，将其替换为更可靠的候选令牌。在多个dMLLMs和多模态推理基准上，ST-Veto始终优于标准的解码策略和先前的VLM推理方法，将准确率提升高达9%，且无需额外训练或生成开销。分析表明，ST-Veto能够将生成过程导向置信度更高、定位更可靠的路径。

## Abstract
Vision Language Models (VLMs) achieve strong reasoning with Chain-of-Thought (CoT) prompting but incur high sequential-generation cost, error accumulation, and limited self-correction. Diffusion Multimodal Large Language Models (dMLLMs) unmask tokens in an order-agnostic process, improving efficiency and enabling iterative refinement, yet their reasoning and how to enhance it remain underexplored. We propose a training-free method, Spatio-Temporal Token Veto (ST-Veto), which leverages the ability to observe all token positions at each diffusion step. Rather than relying only on current-step confidence, ST-Veto vetoes temporally unstable tokens via second-order Taylor prediction of confidence dynamics and filters weakly grounded tokens using image-attention mass, swapping them with safer candidates. Across multiple dMLLMs and multimodal reasoning benchmarks, ST-Veto consistently outperforms standard decoding policies and prior VLM reasoning methods, improving accuracy by up to 9% with no additional training or generation cost. Analyses show that ST-Veto steers generation toward higher-confidence, better-grounded paths.

### 2. 论文提出的方法论

- **核心思想**：**ST-Veto（Spatio-Temporal Token Veto）** 是一种无需训练的、解码时干预方法。它利用dMLLMs在每一步都能观察到所有待解码位置的优势，不再仅依赖当前步的置信度（confidence）来选择待揭开的令牌，而是从时空两个维度评估令牌的可靠性，对不稳定的令牌进行“否决（Veto）”，并用更可靠的候选令牌进行“交换（Swap）”。
- **关键技术细节**：
    - **时间否决（Temporal Veto）**：通过追踪同一位置令牌在多个连续去噪步骤的置信度序列，使用**二阶泰勒展开（Second-order Taylor Expansion）** 来预测下一时刻的期望置信度。如果实际置信度显著低于预测值（产生大的负残差），则判定该令牌在时间上不稳定，将其否决。
        - 公式：`预测置信度 ĉ(t) = c(t-1) + (c(t-1) - c(t-2)) + 1/2 * [(c(t-1) - c(t-2)) - (c(t-2) - c(t-3))]`
        - 不稳定信号：`泰勒残差 ei = c(t)_i - ĉ(t)_i`。
    - **空间否决（Spatial Veto）**：通过计算文本令牌对图像令牌区域的**注意力质量（Image Attention Mass）**，评估其视觉定位性。具体选择模型中间层的注意力矩阵，计算文本令牌分配到图像区域的注意力比例，并求多层的平均值。如果该比例显著低于整体中位数，则判定该令牌视觉定位弱，将其否决。
    - **否决与交换策略（Veto-and-Swap Policy）**：
        1.  **生成提案**：标准方式选出当前置信度最高的前k个令牌作为初始提案集。
        2.  **稳健否决**：对提案集中的令牌，若其时间残差或视觉注意力得分低于各自分布的 `中位数 - 中位数绝对偏差(MAD)`，则标记为“否决”。
        3.  **候选交换**：从非前k名、但满足更严格的“安全”标准（时间和视觉得分均高于 `中位数 + MAD`）且置信度在提案集边界附近的令牌中，选出候选交换集。
        4.  **最终选定**：用候选集中置信度最高的令牌，替换掉原提案集中被否决的令牌。
