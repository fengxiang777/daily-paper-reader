## 摘要
现有的长程LLM智能体记忆系统通常将先前的轨迹检索为被动上下文，而非将其转化为可执行的能力。本文提出MSCE，一种免训练的“记忆-技能协同演化”框架，将智能体经验组织为有依据的步骤轨迹、可复用的程序化策略和声明性环境认知。MSCE将有证据支持且预估收益为正的L2策略固化为可调用技能，这些技能保留证据链接、适用边界、决策指导、验证规则及可靠性估计。它还引入反思加权价值回填，通过密集的局部自我反思传播稀疏的终端反馈，从而产生证据校准的轨迹价值，以调控记忆和技能的演化。在EvoAgentBench和LoCoMo上的实验表明，MSCE显著优于最先进的技能增强和记忆驱动智能体基线，展现出强大的跨域迁移能力和终身演化能力。

## Abstract
Existing memory systems for long-horizon LLM agents often retrieve prior traces as passive context rather than converting them into executable capabilities. In this paper, we propose MSCE, a training-free Memory--Skill Co-Evolution framework that organizes agent experience into grounded step traces, reusable procedural policies, and declarative environmental cognition. MSCE crystallizes evidence-backed L2 policies with positive estimated gain into callable skills that retain evidence links, applicability boundaries, decision guidance, verification rules, and reliability estimates. It further introduces reflection-weighted value backfilling, which propagates sparse terminal feedback through dense local self-reflections to produce evidence-calibrated trace values for governing memory and skill evolution. Experiments on EvoAgentBench and LoCoMo demonstrate that MSCE significantly outperforms state-of-the-art skill-augmented and memory-driven agent baselines, exhibiting strong cross-domain transferability and lifelong-evolution capabilities.

### 2. 论文提出的方法论

-   **核心框架（MSCE - Memory-Skill Co-Evolution）**：一种免训练的记忆-技能协同演化框架，其核心思想是将智能体的交互经验组织成一个可治理、可验证的记忆层级，并基于证据将高价值的行为模式固化为技能。
-   **三层记忆架构（Governed Memory）**：
    -   **L1 轨迹记忆**：存储最基础的、有依据的步骤级证据（状态、动作、观察、自我反思），是一切抽象的基础。
    -   **L2 策略记忆**：从跨任务的L1轨迹中归纳出可复用的程序化策略。每个策略包含触发条件、操作流程、验证规则、适用范围和支持证据。
    -   **L3 环境认知记忆**：从多个L2策略中抽象出关于环境结构和约束的陈述性知识（如实体、规则、约束），为技能执行提供背景信息。
-   **关键技术细节**：
    -   **技能固化（Skill Crystallization）**：将满足两个条件的L2策略“提升”为可调用的技能：1）拥有充分的支持证据且预估增益为正；2）近期表现稳定，无需大幅修订。固化生成的技能卡不仅包含操作流程，还保留证据锚点、决策指导、验证规则和基于平滑成功率的可靠性评分，兼顾可追溯性与可靠性。
    -   **反思加权价值回填（Reflection-Weighted Value Backfilling）**：
        -   **目的**：解决任务终端反馈信号通常稀疏、延迟，而步骤级自我反思虽稠密但可靠性存疑的矛盾。该机制旨在对每一步**轨迹进行价值校准**。
        -   **技术细节**：利用一个专门的LLM评分器，对每一步的自我反思进行打分（α），衡量其忠实性、因果洞察力、可迁移性等。然后，根据以下公式将稀疏的终端奖励 `R_i` 反向传播到每一个步骤 `t`，计算出每个步骤的证据校准轨迹价值 `V`：
            `V(f_i,t) = α_{i,t} * R_i + (1 - α_{i,t}) * γ * V(f_i,t+1)`
            其中，`γ` 是折扣因子。高质量反思的步骤会获得更接近于终端反馈的价值，而低质量反思的步骤则更多地继承后续步骤的价值。
        -   **作用**：这些校准后的价值信号是驱动**L1检索、L2策略归纳、L3环境认知抽象和技能固化**的统一信号源泉，实现了记忆与技能的闭环协同演化。
