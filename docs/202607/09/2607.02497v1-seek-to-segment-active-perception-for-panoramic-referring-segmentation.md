## 摘要
现有的指代分割模型被动地处理从固定视角捕获的静态图像，限制了它们在具身人工智能中的适用性，其中智能体必须在连续的360°环境中执行主动感知。为了弥补这一差距，我们引入了一项新任务：主动全景指代分割（APRS）。在此设定下，智能体需要调整其观察方向（Δθ, Δφ）来探索360°环境，寻找用户指令中指定的对象进行分割。为了解决这一具有挑战性的任务，我们提出了PanoSeeker，一种用于高效APRS的记忆增强型智能体。PanoSeeker并非依赖启发式扫描，而是将视觉-语言模型（VLM）与EgoSphere——一种显式的空间视觉记忆——相结合。通过逐步将连续的局部观察整合到统一的360°表征中，EgoSphere使智能体能够规划高效且无冗余的搜索轨迹。一旦找到目标，智能体将执行主动视角对齐并输出分割掩膜。此外，我们精心整理了一个由专家标注的、带有记忆时间线的搜索轨迹数据集，用于监督微调，随后进行强化学习后训练，以显式优化PanoSeeker的探索效率。在我们新建立的APRS基准上的广泛实验表明，PanoSeeker实现了卓越的搜索效率和分割精度，显著超越了适配后的最先进基线。

## Abstract
Existing referring segmentation models passively process static images captured from fixed perspectives, limiting their applicability in Embodied AI, where agents must perform active perception in the continuous 360$^\circ$ environments. To bridge this gap, we introduce a novel task: Active Panoramic Referring Segmentation (APRS). In this setting, an agent is required to adjust its viewing direction ($Δθ, Δφ$) to explore the 360$^\circ$ environment, seeking the object specified by a user instruction for segmentation. To tackle this challenging task, we propose PanoSeeker, a memory-augmented agent for efficient APRS. Rather than relying on heuristic scanning, PanoSeeker integrates a Vision-Language Model (VLM) with EgoSphere, an explicit spatial visual memory. By progressively integrating sequential local observations into a unified 360$^\circ$ representation, EgoSphere enables the agent to plan efficient and non-redundant search trajectories. Once the target is found, the agent performs active viewpoint alignment and outputs the segmentation mask. Furthermore, we curate an expert-annotated search trajectory dataset with memory timelines for Supervised Fine-Tuning, followed by Reinforcement Learning post-training to explicitly optimize PanoSeeker's exploration efficiency. Extensive experiments on our newly established APRS benchmark demonstrate that PanoSeeker achieves superior search efficiency and segmentation accuracy, significantly outperforming adapted state-of-the-art baselines.

- **核心问题**：如何设计一个能够根据自然语言指令，在完整全景空间内主动调整观察方向、高效寻找目标并完成像素级分割的智能体？
- **提出任务**：本文首次定义并构建 **主动全景指代分割（Active Panoramic Referring Segmentation, APRS）** 任务。该任务要求智能体从初始视角出发，通过调整相机朝向 \((\Delta\theta, \Delta\phi)\) 探索360°环境，最终找到指令指定的物体并输出分割掩膜。
- **挑战与意义**：克服了传统方法因几何畸变、全局上下文丢失或三维重建代价高而无法胜任主动多视角感知的困境，为具身场景下的视觉理解提供了一个可扩展、低成本的评测基准。

## 2. 方法论

- **整体框架**：提出 **PanoSeeker**，一种结合视觉‑语言模型（VLM）与显式空间记忆 **EgoSphere** 的主动感知智能体。
- **EgoSphere 空间视觉记忆**
  - 将360°环境离散化为一张等距柱状投影（ERP）画布 \(\mathbf{M} \in \mathbb{R}^{H\times W\times 3}\)，初始全零。
  - 每一步将当前局部视野 \(\mathbf{V}_t\) 通过逆向球面投影更新至画布：\(\mathbf{M}_t = \mathbf{M}_{t-1} \oplus \text{Proj}^{-1}(\mathbf{V}_t, \theta_t, \phi_t, \psi)\)。未被观察的区域保持为零，形成明确的“待探索”前沿。
  - 在记忆画布上叠加空间提示：当前视野中心十字准星、经纬度网格（30°间隔）以及历史搜索轨迹箭头，将搜索问题转化为视觉上的“填空”任务。
- **决策与控制**
  - PanoSeeker 以 VLM（Qwen3-VL-8B）为核心，同时接收当前观察 \(\mathbf{V}_t\)、带记忆标注的画布 \(\mathbf{M}_t\) 以及指令 \(I\)，预测下一个动作 \(\mathbf{a}_t \in \{(\Delta\theta_t, \Delta\phi_t), \texttt{[STOP]}\}\)。
- **两阶段训练**
  - **监督微调（SFT）**：利用专家标注的搜索轨迹数据集，最小化动作的负对数似然：\(\mathcal{L} = -\mathbb{E}_{\mathcal{T}\sim\mathcal{D}_{\text{expert}}} \sum_{t=1}^T \log p_\theta(\mathbf{a}_t | \mathbf{V}_t, \mathbf{M}_t, I)\)，使模型学会合理视角调整。
  - **强化学习（RL）**：采用 GRPO 算法进一步优化探索效率。奖励由两部分组成：
    - 效率奖励 \(R_{\text{eff}} = \gamma [\text{dist}(\mathbf{V}_{t-1}, \mathbf{V}^*) - \text{dist}(\mathbf{V}_t, \mathbf{V}^*)] - \eta\)，鼓励沿最短路径接近目标并惩罚冗余移动。
    - 终端奖励 \(R_{\text{term}}\)：成功找到目标（视角距离小于阈值）时给予基础分 + IoU 加权分，否则给予惩罚。
- **主动对齐与分割**：当智能体发出停止信号后，利用 VLM 的定位能力预测目标边界框，再通过 SAM-3 进行中心校正、跟踪与分割，最终输出稳定的分割掩膜。

## 3. 实验设计

- **数据集与基准**：自建 **APRS 基准**，融合 360-Indoor、PANDORA、SUN360 三个来源，涵盖 4,971 个场景、7,420 个样本（7:3 划分训练/测试）。指令包含四种类型：以自我为中心（EGO）、唯一属性（UNIQ）、以物体为中心（ALLO）以及多跳推理（MULTIHOP），以全面评测空间推理能力。
- **评估指标**：成功率 SR（最终视角偏差 < 阈值且 IoU ≥ 0.5）、平均步数 AS、路径长度加权成功率 SPL、平均 IoU（mIoU）。
- **对比方法**：
  - 静态方法：直接对整幅全景图进行指代分割的模型（VLT、CRIS、LISA、SAM4MLLM、VisionReasoner）。
  - 启发式扫描：按固定顺序扫描上半球、赤道、下半球，直到模型预测出掩膜。
  - VLM 智能体：以文本日志为记忆的主动探索智能体（Qwen3-VL-30B、Gemini-3-Flash、GPT-5.2）。
- **主要结果**：PanoSeeker 在所有指标上均大幅领先，成功率 75.4%，mIoU 55.8%，SPL 0.57（比 GPT-5.2 提升约 58%），仅需 4.8 步。

## 6. 主要结论与发现

- EgoSphere 显式空间记忆能够将顺序局部观察整合为连续的全景画布，有效防止重复探索，显著提升搜索效率。
- 监督微调使智能体具备基本的搜索能力，强化学习进一步将其优化为接近最短路径的高效探索策略。
- PanoSeeker 在新建立的 APRS 基准上，以更少的搜索步数实现了更高的成功率和分割精度，大幅超越传统的静态分割方法和基于文本日志的 VLMs 智能体，为具身主动感知提供了新的强基线。

## 7. 优点

- **任务创新性**：首次定义 APRS，将指代分割从静态单帧扩展至主动全景探索，贴合具身智能需求。
- **记忆设计巧妙**：EgoSphere 将时间序列感知转化为空间填空问题，固定分辨率画布避免了随步数增长带来的上下文膨胀。
- **训练策略有效**：SFT 提供基本行为规范，GRPO 强化学习直接优化路径效率，两者结合使性能显著提升。
- **基准构建细致**：数据集规模较大，涵盖四种空间指代类型，并建立了全面的评估协议。
- **实验严谨**：与多类方法对比，消融充分，同时提供了可视化轨迹分析。

## 8. 不足与局限

- **仿真与现实的差距**：环境完全由全景图像投影模拟，未涉及真实机器人感知噪声、遮挡动态变化、行动执行误差等真实世界因素。
- **动作粒度**：动作空间离散为 30° 的整数倍，可能无法满足高精度对准需求；探索效率最优时 AS 仍达 4.8，理论上最短路径可能更短。
- **方法覆盖面**：PanoSeeker 仅基于单一 VLM（Qwen3-VL），未探讨更换不同 VLM 的影响；基准也未包含多目标、无目标或复杂动态场景。
- **成本与可复现性**：专家标注搜索轨迹成本较高，虽然提供了 RL 后训练，但完全脱离专家轨迹的纯 RL 训练效果未知。
- **终止条件单一**：搜索终止仅依赖 [STOP] 预测与距离阈值，未考虑目标部分遮挡或被其他物体暂时挡住时的重识别策略。
