## 摘要
持续后训练正成为适应视觉-语言模型到不断演变任务的核心范式。近期工作越来越倾向于强化学习而非监督微调，这源于一个信念：强化学习本质上更不易遗忘。然而，这一信念尚未得到充分验证，因为现有证据大多来自过时或同质的基准。为了重新审视这一假设，我们提出了 MRCL，一个基于多样且近期发布的多模态数据集构建的多模态推理持续学习基准。在 MRCL 上的实验表明，强化学习在持续后训练中仍可能遭遇严重的灾难性遗忘。为了解决这一挑战，我们提出了持续策略优化（CPO），一个基于先前任务行为 KL 目标的无回放框架。CPO 使用理论上合理的参数移动正则化来限制策略在先前任务上的漂移。跨多个模型规模的广泛实验表明，CPO 能持续减少遗忘，同时保留甚至在某些情况下提升预训练模型的能力。在 Qwen3-VL-8B 上，CPO 将遗忘减少了 13.7%，并将预训练能力提升了 7.0%。实现代码可在 https://github.com/MaolinLuo/CPO 获取。

## Abstract
Continual post-training is becoming a central paradigm for adapting vision-language models to evolving tasks. Recent work has increasingly favored reinforcement learning over supervised fine-tuning, driven by the belief that reinforcement learning is inherently less prone to forgetting. However, the belief remains insufficiently validated, as existing evidence is largely drawn from outdated or homogeneous benchmarks. To revisit this assumption, we introduce MRCL, a Multimodal Reasoning Continual Learning benchmark built from diverse and recently released multimodal datasets. Experiments on MRCL show that reinforcement learning can still suffer from severe catastrophic forgetting during continual post-training. To address this challenge, we propose Continual Policy Optimization (CPO), a replay-free framework grounded in the prior-task behavioral KL objective. CPO uses a theoretically justified parameter-movement regularization to limit policy drift on previous tasks. Extensive experiments across multiple model scales demonstrate that CPO consistently reduces forgetting while preserving, and in some cases improving, pretrained model capabilities. On Qwen3-VL-8B, CPO reduces forgetting by 13.7\% and improves pretrained capability by 7.0\%. The implementation code is available at https://github.com/MaolinLuo/CPO.

- **整体含义**：本文通过构建一个全新、多样化的多模态推理持续学习基准 MRCL，揭示了主流 RL 方法在持续后训练中同样会发生严重灾难性遗忘，挑战了“RL 天然健忘”的假设。并据此提出无回放的持续策略优化算法 CPO，为安全、高效的 VLM 持续强化学习提供了理论支撑和实用方案。

## 2. 论文提出的方法论
- **核心思想**：将灾难性遗忘归因于策略在旧任务数据上的行为 KL 散度（即“旧任务行为 KL”），通过约束该 KL 来保留先前知识。因旧数据不可访问，作者将 KL 约束松弛为参数空间上的移动正则化，并利用“参数在训练中的累积移动量”作为参数重要性的梯度无关代理，避免计算昂贵的 Fisher 信息矩阵。
- **关键技术细节**：
  - **旧任务回归界**：证明旧任务期望回报的下降可由旧策略前缀访问分布下的平均 token 级 KL 散度上界控制。
  - **二阶近似与 Fisher 对角松弛**：将旧任务行为 KL 在参数空间中作二阶泰勒展开，得到带 Fisher 信息矩阵对角元的平方惩罚项，等价于根据参数敏感度加权的参数移动正则化。
  - **梯度无关的移动代理**：理论证明，在局部平稳 SGD 下，参数移动幅度期望正比于 Fisher 对角元，从而可以用任务训练期间的参数-移动量作为重要性的排序依据，无需保存旧数据或计算额外梯度。
  - **持续策略优化（CPO）算法**：
    - 维护一个累积的保护参数集 \\(S^{(i-1)}\\)（由历史各任务的 top \\(p\%\\) 移动参数组成）。
    - 训练第 \\(i\\) 个任务时，在 RL 损失上添加一个保护集上的 L1 正则化项：\\( \lambda \cdot \frac{1}{\|\mathbf{m}^{(i-1)}\|_0} \|\mathbf{m}^{(i-1)} \odot (\theta - \tilde{\theta})\|_1\\)，以稀疏方式将重要参数拉向旧模型。
    - 任务结束后，将当前任务中移动最大的 \\(p\%\\) 参数加入保护集。
- **算法流程**（文字描述）：
  1. 初始化保护集 \\(S^{(0)} = \varnothing\\) 和检查点 \\(\theta^{(0)}\\)。
  2. 对序列中的每一任务 \\(T_i\\)：
     - 以 \\(\tilde{\theta} = \theta^{(i-1)}\\) 为旧模型，用带保护集 L1 正则化的 RL 目标优化 \\(\theta\\)。
     - 训练结束存储 \\(\theta^{(i)}\\)。
     - 计算参数移动量，把 top \\(p\%\\) 纳入 \\(S^{(i)}\\)。
  3. 重复直到所有任务完成。

## 6. 论文的主要结论与发现
- **RL 也会遗忘**：在 MRCL 基准上，GRPO、GSPO 等现代 RL 算法同样遭受严重的灾难性遗忘，先前的“RL 天然不易遗忘”假设不成立。
- **CPO 有效抑制遗忘**：提出的 CPO 在所有模型尺度上均显著降低遗忘，同时保持良好的新任务学习能力（塑性）。在 Qwen3‑VL‑8B 上，MFN 相比 GSPO 提升 13.7%，预训练能力平均提升 7.0%。
- **L1 正则化更优**：保护集上的 L1 正则化产生稀疏的参数移动，多数重要参数几乎保持不变；而同位置的 L2 正则化导致密集漂移，效果更差。
- **通用能力保留**：CPO 不仅不破坏预训练能力，反而在多个数学、文档理解任务上超越原始模型，说明参数移动正则化能稳定保留甚至提升广泛技能。

## 7. 优点
- **理论驱动、目标明确**：从旧任务行为 KL 出发，严格推导出参数移动正则化形式，给出了遗忘的上界，使方法具有理论支撑。
- **无回放、低成本**：CPO 完全不需要存储历史数据，仅基于训练中的参数移动量选择重要参数，避免了巨大的计算和存储开销，易于扩展至大模型。
- **简单实用的算法**：算法流程清晰，只在原有 RL 损失上增加一个稀疏 L1 项，实现复杂度低。
- **新的基准 MRCL**：利用近期多模态数据构建基准，有助于减少预训练污染，为持续学习研究提供了更可靠的评估平台。
- **充分的实验和可视化**：多模型、多基线、多下游任务评估，辅以热力图、损失曲线和案例分析，结果说服力强。

## 8. 不足与局限
- **任务数量有限**：MRCL 仅包含 5 个任务，任务类型未覆盖生成、对话等更开放的形式，持续学习长度较短。
- **任务顺序固定**：未探讨不同任务顺序对遗忘和方法性能的影响。
- **仅基于 terminal reward**：理论推导和实验均限定在终端奖励的 RL 设定，未涉及过程奖励或偏好优化等更复杂的 RL 范式。
