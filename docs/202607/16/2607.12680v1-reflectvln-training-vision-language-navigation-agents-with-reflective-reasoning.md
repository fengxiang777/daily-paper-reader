## 摘要
现有的视觉语言导航方法通常将VLM与航点解码器耦合以生成多步行动规划，但通常缺乏一个明确的闭环机制来跟踪语义进展、诊断执行失败并从长距离导航中的误差累积中恢复。为解决这一问题，我们提出了ReflectVLN，一个通过双向交互的意图和执行智能体组织决策的智能体式VLN框架。意图智能体进行子任务分解与反思，生成可执行的子任务描述作为纠正计划。以这些描述为条件，执行智能体在当前观测下将其转化为短距离动作，同时监控子目标进度并检测偏离行为。至关重要的是，ReflectVLN实现了闭环双向通信：执行智能体发出进度和偏离信号，以便按需触发反思和子任务更新，意图智能体则返回结构化指导，重新条件化后续动作以实现恢复。为了鼓励具有可解释中间推理的时间连贯决策，我们引入了Action Chain-of-Thought (Action-CoT)，一种路径条件的双查询训练方案用于动作生成。在标准VLN基准上的实验表明，在有限的数据预算下，ReflectVLN提高了成功率和路径效率，具有较低的训练成本和推理时较少的高层意图调用，同时为分析和协作提供了可解释的中间决策。代码见：https://github.com/AIprogrammer/ReflectVLN

## Abstract
Existing vision-language navigation methods often couple a VLM with waypoint decoders to produce multi-step action plans, but they typically lack an explicit closed-loop mechanism for tracking semantic progress, diagnosing execution failures, and recovering from error accumulation in long-horizon navigation. To address this gap, we propose ReflectVLN, an agentic VLN framework that organizes decision-making through bidirectionally interactive intention and execution agents. The intention agent performs subtask decomposition and reflection, generating executable subtask descriptions as corrective plans. Conditioned on these descriptions, the execution agent grounds them into short-horizon actions under current observations while monitoring sub-goal progress and detecting off-track behavior. Crucially, ReflectVLN enables closed-loop bidirectional communication: the execution agent emits progress and deviation signals to trigger reflection and subtask updates on demand, and the intention agent returns structured guidance that reconditions subsequent actions for recovery. To encourage temporally coherent decisions with interpretable intermediate rationales, we introduce Action Chain-of-Thought (Action-CoT), a path-conditioned dual-query training scheme for action generation. Experiments on standard VLN benchmarks show that ReflectVLN improves success rates and path efficiency under a constrained data budget, with favorable training cost and fewer high-level intention calls at inference time, while providing interpretable intermediate decisions for analysis and collaboration. ### 1. 论文的核心问题与整体含义（研究动机和背景）
本论文聚焦于**连续环境中的视觉语言导航（VLN）** 任务，即智能体需遵循自然语言指令在复杂三维场景中导航至目标。当前主流方法大多将视觉语言模型（VLM）与航点解码器耦合，形成端到端策略，存在一个关键缺陷：
*   **缺乏闭环反思与恢复机制**：现有分层式设计（慢-快系统）多为单向通信，高层规划模块制定子目标并下发给执行器，但执行过程中的**语义进展、执行偏差和失败信号无法系统性地反馈给规划模块**进行及时修正。
*   **长距离误差累积问题**：在长距离导航中，微小的控制误差会不断累积，导致智能体偏离路线，最终任务失败。

因此，本研究的核心动机是**构建一个具有显式反思能力的智能体框架**，通过双向交互实现高层规划与低层执行的闭环协同演化，从而在长距离导航中主动诊断错误、修正计划并实现可靠恢复。

### 2. 论文提出的方法论
ReflectVLN 的核心思想是将决策过程组织为**双向交互的意图智能体（Intention Agent）和执行智能体（Execution Agent）**。

*   **核心思想与架构**:
    *   **双智能体解耦**: 意图智能体负责高层次的语义理解、子任务分解和反思性规划；执行智能体负责将子目标转化为短距离连续控制。两者使用独立的Qwen2.5-VL-3B模型参数。
    *   **闭环双向通信**: 执行智能体在执行动作的同时，会发出一个状态令牌。此令牌充当从执行到意图的**反馈接口**，触发意图智能体按需更新，构成了“意图-执行-反馈-再规划”的闭环。

*   **关键技术细节**:
    *   **反射式数据构建**: 设计了一套单轮数据增强流水线，将模型的失败轨迹转化为反思性训练样本。流程包括：① 执行当前策略收集失败案例；② 调用专家生成纠正轨迹，并由VLM生成自然语言的反思性描述；③ 通过一致性验证过滤幻觉和不一致样本，最终生成带反思反馈的纠正轨迹数据。
    *   **执行智能体与 Action-CoT**: 论文提出 **Action Chain-of-Thought**，一种路径条件的双查询训练方案。执行智能体在预测短距离动作序列 $(\hat{A}_t)$ 前，会先预测一个更粗粒度的未来路径 $(\hat{P}_t)$，然后将此路径表示作为条件指导后续动作生成。这增强了局部控制与全局路径的一致性。
    *   **状态令牌机制**: 执行智能体在每一步输出三种状态令牌之一：
        *   `<VLNBOA>` ：继续执行动作。
        *   `<VLNBOR>` ：表示当前子目标完成，请求意图智能体生成下一个常规子目标。
        *   `<VLNBOC>` ：检测到偏离路径，请求意图智能体进行诊断并生成纠正性子目标。
    *   **训练流程**: 采用两阶段训练。第一阶段对执行智能体进行基于专家轨迹的Action-CoT预训练；第二阶段使用专家数据和反思性数据进行状态感知微调，同时训练意图智能体的语言生成和执行智能体的航点及状态令牌预测。

### 7. 优点
*   **创新的闭环交互范式**：明确的双向反馈接口使高层规划与低层执行能动态协同，是相较于一众“开环”或“单向”分层方法的核心优势。
*   **高效的数据利用**：设计的反射性数据构建流水线能有效地将失败经验转化为训练信号，在小数据预算下实现了高收益。
*   **可解释性**：通过子目标语言描述和Action-CoT的中间路径预测，模型的决策过程更加透明，便于分析和人机协作。
*   **任务效能与推理效率的平衡**：事件驱动的意图调用机制在维持高任务成功率的同时，减少了对昂贵高层模型的请求次数。

### 8. 不足与局限
*   **评估局限性**: 实验仅在两个主流的室内导航基准（R2R / RxR）中进行，其泛化能力在其他环境（如室外、更动态的场景）中尚待验证。
*   **对比偏差**: 虽然与CorrectNav进行了对比，但只取了其“第一轮”自我纠正的结果，未与其完整的“多轮飞轮”优化结果进行对比。同时，文中也承认解耦模型和统一模型在总参数量上不同（6B vs 7B）。

*   **应用限制**: 方法严重依赖VLM的推理和语言生成能力，其性能可能受限于基础模型的能力上限。此外，尽管数据效率高，但两阶段训练和双模型架构仍然带来了一定的工程复杂性。
