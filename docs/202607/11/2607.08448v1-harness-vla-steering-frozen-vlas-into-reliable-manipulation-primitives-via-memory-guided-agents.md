## 摘要
语言条件操作需要精确的接触丰富控制和在语言、场景及长时间范围内的稳健推理。端到端的视觉-语言-动作（VLA）模型提供了强大的局部视觉运动技能，但它们是在分布内的任务轨迹上训练的，在部署扰动下（例如语义重定向、目标重新绑定、空间布局变化以及不稳定的局部接触）经常失败。LLM编码代理提供了互补的语义和组合推理，但纯粹的分析原语在处理不规则抓取、受限放置和铰接物体交互方面存在困难。我们提出了Harness VLA，一个记忆增强的代理框架，它将一个冻结的VLA暴露为一个可重试的接触丰富原语，并将其与一个小型的固定分析原语库组合，用于定位、暂存、运输、导航和释放。该框架不是扩展技能库，而是从特定任务的执行轨迹、全局成功规则和故障模型中学习这些固定原语的操作范围。通过将语义重新定位、非接触执行和VLA重新暂存提升到规划器，同时将冻结的VLA保留用于局部接触丰富阶段，Harness VLA在不进行微调的情况下将预训练的VLA扩展到其原始的轨迹分布之外。在受到扰动的桌面、家用厨房以及从干净到随机化的双臂操作任务中，Harness VLA在LIBERO-Pro和RoboCasa365上分别比最强的相关基线提高了38.6和25.4个百分点，并在RoboTwin C2R上达到了58.4%的成功率。

## Abstract
Language-conditioned manipulation requires both precise contact-rich control and robust reasoning over language, scenes, and long horizons. End-to-end Vision-Language-Action (VLA) models provide strong local visuomotor skills, but they are trained on in-distribution task trajectories and often fail under deployment perturbations such as semantic retargeting, goal re-binding, spatial-layout shifts, and unstable local contacts. LLM coding agents provide complementary semantic and compositional reasoning, but purely analytic primitives struggle with irregular grasping, constrained placement, and articulated-object interaction. We present Harness VLA, a memory-augmented agentic framework that exposes a frozen VLA as a retryable contact-rich primitive and composes it with a small fixed library of analytic primitives for grounding, staging, transport, navigation, and release. Rather than expanding the skill library, the harness learns the operating range of these fixed primitives from task-specific execution traces, global success rules, and failure models. By lifting semantic re-grounding, non-contact execution, and VLA re-staging to the planner while reserving the frozen VLA for local contact-rich phases, Harness VLA extends pretrained VLAs beyond their original trajectory distribution without finetuning. Across perturbed tabletop, household kitchen, and clean-to-randomized bimanual manipulation, Harness VLA improves over the strongest relevant baselines by 38.6 and 25.4 percentage points on LIBERO-Pro and RoboCasa365, respectively, and reaches 58.4% on RoboTwin C2R.

## 1. 论文的核心问题与整体含义

-   **核心问题**：语言条件机器人操作任务面临双重挑战。端到端的视觉-语言-行动（VLA）模型在接触丰富的局部控制上表现强大，但在部署环境发生扰动（如语义重定向、目标物体位置变化、空间布局改变）时容易失败，无法有效遵循新的语言指令。另一方面，基于大型语言模型（LLM）的编码代理具备良好的语义推理和长程任务组合能力，但其使用的分析性原语（如确定性运动控制）难以处理不规则抓取、精密放置等接触丰富的物理交互。
-   **整体含义**：本文旨在解决这两种范式的局限性。核心思想是**不进行模型微调，而是通过一个记忆增强的智能代理框架（Harness VLA）将冻结的预训练VLA与分析性原语库有机结合起来**。该框架让 LLM 规划器负责语义理解和任务编排，仅在需要物理接触技能时调用冻结的 VLA，从而将 VLA 从处理长程任务的全能策略转变为专注局部接触的可靠操作原语，显著提升系统在分布外和扰动环境下的鲁棒性和泛化能力。

## 2. 论文提出的方法论

-   **核心思想**：提出一个非对称的分层框架 **Harness VLA**，它将任务执行分解为**高层规划**与**低层执行**。高层是一个由记忆引导的 LLM 代理规划器，负责语义解析、场景理解和任务编排；低层是一个固定的原语库，包含分析性原语和一个特殊的 `VLA_ACT` 原语，用于调用冻结的 VLA。
-   **关键组件与技术细节**：
    -   **智能体执行循环（Agentic Execution Loop）**：采用一种“读取-求值-打印循环”（REPL）风格的交互机制。规划器接收语言指令 `ℓ` 和当前的多模态观察 `o_t`（RGB图像、深度图和本体状态），选择一个原语 `c_t`，通过 JSON 命令发送给执行器。执行器在物理引擎中执行原语直到内部后置条件满足，然后返回新的观察 `o_{t+1}`。循环重复，直到任务完成或超出步数预算。
    -   **记忆系统**：
        -   **任务特定记忆（Task Specific Memory）**：存储从单个参考任务实例（种子0）自主探索出的成功解决方案。它包含一个参数化的 JSONL 命令序列（用符号化的感知查询替代具体坐标，使其在不同空间布局下可重用）和一段总结策略、失败模式的 JSON 摘要。在部署时，规划器检索记忆并动态重定位场景，从而在扰动环境下少样本（few-shot）地执行任务。
        -   **全局记忆（Global Memory）**：存储跨任务的可重用成功规则（如VLA的最佳使用场景）和失败模型（如识别空抓取或虚假成功）。这个经验库帮助规划器避免在不同任务中重复历史错误。
    -   **统一原语接口（Unified Primitive Interface）**：系统将所有的底层控制手段（包括冻结VLA和确定性控制器）统一为`P` 里一组固定的原语。规划器不生成低层动作，而是选择原语并绑定参数。原语库保持小规模且固定，部署时不能创建新原语。
        -   **分析性原语**：基于模型的控制指令，用于非接触丰富的结构化操作。包括组合原语（如`MOVE_TO`移动末端执行器、`MOVE_POSE`调整位姿）和原子原语（如`ROTATE_WRIST`旋转手腕、`SET_GRIPPER`控制夹爪）。
        -   **VLA原语（`VLA_ACT`）**：封装了一个冻结的预训练VLA模型。规划器提供任务提示和提前终止条件 `τ`，VLA 模型则在局部生成动作块，执行抓取、按压、插入等接触丰富交互，直到条件满足或达到预算上限。
-   **算法流程**：整个流程分为两个阶段。在**探索引导阶段**，代理在单个参考任务实例上通过反复试验发现可行解决方案，并将成功和失败经验结构化存入记忆。在**部署评估阶段**，代理在未见过的扰动环境中，严格依据记忆和实时感知，按序调用原语执行任务。

-   **仿真到现实的鸿沟（Sim-to-Real Gap）**：所有实验均在仿真环境（MuJoCo）中进行，尚未在真实物理机器人上验证，框架在真实世界传感器噪声、不可见动态场景下的表现仍有待观察。
-   **开环反馈与依赖顶层LLM**：高层规划器和低层 VLA 之间是开环的指令-执行关系。系统的整体推理速度、延迟和成本严重依赖于所使用的 LLM API 的性能和费用。
-   **对预训练模型的依赖性**：系统性能的上限受到冻结 VLA 的接触技能和 LLM 的视觉推理能力的直接约束。如果 VLA 完全无法完成某个接触子任务（如一些需要极高频力反馈的操作），框架也无法成功。
-   **复杂长程任务的局限性**：论文结论中明确指出，在高度杂乱、长程的任务中，由于缺乏细粒度的图像描述，系统的结构推理能力受限。
-   **实验覆盖的可能不足**：尽管已有的实验量很大，但主要专注于单步扰动。对于更加动态、连续变化的环境干扰或需实时调整的任务，其鲁棒性尚未得到测试。
